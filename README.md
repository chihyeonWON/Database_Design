## 데이터베이스
```
조직에 필요한 정보를 얻기 위해 논리적으로 연관된 데이터를 모아 구조적으로 통합해 놓은 것이다.
```

## 데이터베이스의 개념
```
데이터에 대한 통합된 데이터, 저장된 데이터, 운영 데이터, 공용 데이터를 말한다.
```
